	Поставленная задача: 
Построить парсер возвращающий информацию с сайта film.ru о выходе новых серий аниме сериалов каждый день и отправляющий эту информацию в виде excel таблицы в телеграмм бот.  

	Инструкция по сборке и запуску кода:
	
Код работает автоматически без ввода данных.

	- Получаем HTML-контент с указанного URL.
	
	- Сохраняем полученный HTML в файл 
	
	- Парсим HTML с помощью BeautifulSoup 
	
	- Получаем количество записей па странице
	
	- Создаем пустой список, в котором будут храниться записи об оценках в виде словарей
	
	- С помощью функции 'spisok' находим название сериала и вышедшею серию 
	
	- Сохраняем подученные данные в переменной 'dat'
	
	- Осуществляем запись в файл 'New_anime.xlsx'
	
	- Отправляем этот файл в телеграмм-бот с помощью функции 'send_file_via_telegram'
	

Функцию 'spisok' допускает использования для парсера других разделов сайта  film.ru.

Функции 'send_file_via_telegram' допускает настройку отправки данных в другой телеграмм-бот
